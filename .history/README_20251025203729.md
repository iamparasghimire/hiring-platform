## ğŸ“¦ Installation

### Prerequisites

- Python 3.12+
- Node.js 20+
- npm/yarn
- Git

### Detailed Backend Setup

1. **Clone repository**
   ```bash
   git clone <repository-url>
   cd hiring/hiring_platform
   ```

2. **Create virtual environment**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run migrations**
   ```bash
   python3 manage.py migrate
   ```

5. **Create admin user**
   ```bash
   python3 manage.py createsuperuser
   # Enter: username, email, password
   ```

6. **Start server**
   ```bash
   python3 manage.py runserver 0.0.0.0:8000
   ```

### Detailed Frontend Setup

1. **Navigate to frontend**
   ```bash
   cd frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - http://localhost:3000

## ğŸ“š API Documentation

### Authentication Endpoints

#### Company Registration
```http
POST /api/company-register/

{
  "name": "Company Name",
  "email": "admin@company.com",
  "password": "SecurePassword123",
  "location": "San Francisco",
  "industry": "Technology",
  "website": "https://company.com"
}
```

**Response** (201 Created):
```json
{
  "token": "9f9cecba8ee7762c7c88f376ca291bf824f1b236",
  "company_id": 4,
  "company_name": "Company Name",
  "email": "admin@company.com"
}
```

#### Company Login
```http
POST /api/company-login/

{
  "email": "admin@company.com",
  "password": "SecurePassword123"
}
```

### Job Endpoints

#### List Jobs (with filters)
```http
GET /api/jobs/?search=developer&job_type=full_time&experience_level=senior&salary_min=80000
```

**Query Parameters:**
- `search` - Job title/description search
- `job_type` - full_time, part_time, contract, temporary
- `experience_level` - entry, mid, senior
- `category` - Category ID or slug
- `location` - Location filter
- `salary_min` - Minimum salary
- `salary_max` - Maximum salary
- `page` - Page number

#### Get Job Details
```http
GET /api/jobs/{id}/
```

#### Create Job (Authenticated)
```http
POST /api/jobs/
Authorization: Token {token}

{
  "title": "Senior Python Developer",
  "location": "San Francisco",
  "description": "Job description...",
  "requirements": "- 5+ years experience\n- Django knowledge",
  "salary_min": 120000,
  "salary_max": 160000,
  "job_type": "full_time",
  "experience_level": "senior",
  "category": 1,
  "skills_required": "Python, Django, REST API",
  "benefits": "- Health insurance\n- Remote work"
}
```

### Application Endpoints

#### Submit Application
```http
POST /api/applications/

{
  "job": 1,
  "candidate_name": "John Doe",
  "candidate_email": "john@example.com",
  "candidate_phone": "+1-555-1234",
  "cv": <file>,
  "cover_letter": "Optional"
}
```

**File Upload Requirements:**
- Formats: PDF, DOC, DOCX, TXT
- Max Size: 5MB

#### List Applications
```http
GET /api/applications/?job={id}&status=pending
```

#### Update Application
```http
PATCH /api/applications/{id}/
Authorization: Token {token}

{
  "status": "reviewing",
  "rating": 4,
  "notes": "Strong candidate"
}
```

#### Get Company Jobs

## ğŸ› Troubleshooting

### "Cannot connect to localhost:8000"
```bash
# Check if Django is running
curl http://localhost:8000/api/jobs/

# Start Django server if needed
cd hiring_platform
python3 manage.py runserver 0.0.0.0:8000
```

### "Cannot connect to localhost:3000"
```bash
# Start Next.js if not running
cd frontend
npm run dev

# Or use different port if already in use
npm run dev -- -p 3001
```

### "401 Unauthorized on Login"
- Verify email and password are correct
- Check if user exists in database
- Ensure Token model is migrated: `python3 manage.py migrate`

### "CV upload fails"
```bash
# Ensure media folder exists
mkdir -p media/cvs
chmod 755 media
```

### "Module not found" errors in frontend
```bash
# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### "Foreign key constraint error"
```bash
# Ensure all migrations are applied
python3 manage.py migrate

# Check database consistency
python3 manage.py check
```
